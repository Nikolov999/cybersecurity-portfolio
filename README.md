## Projects

This portfolio includes several security-focused tools and research projects:

### EchoPentest Toolkit
A growing set of security tools focused on practical defensive visibility and assessment:

- **EchoExposure**  
  External attack surface monitoring and domain exposure analysis.

- **EchoVuln**  
  Endpoint vulnerability visibility and prioritization with agent-based snapshots.

- **EchoPhishing (in development)**  
  Email threat detection and phishing analysis platform designed for small and mid-size environments.

These tools are designed with usability, automation, and realistic threat modeling in mind, with the long-term goal of supporting small and medium organizations that lack dedicated security teams.
