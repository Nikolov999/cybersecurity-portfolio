AI Phishing Analyzer Prototype — EchoPentest

An explainable phishing detection prototype designed for portfolio demonstration. The goal is to show an analyst-style workflow: input parsing, feature extraction, scoring/classification, and clear reasoning outputs.

Prototype Explainable Output Feature Signals Offline-Friendly
View Repository Project README

Core Workflow

  • Input: suspicious email content (headers/body), URLs, and indicators
  • Signals: domain mismatch, urgent language, link obfuscation, spoof indicators
  • Scoring: weighted risk model producing a final classification
  • Explainability: per-signal breakdown for analyst review

What This Demonstrates

Skill Focus
Detection logic, feature engineering, and analyst reporting UX
Output
Classification + rationale + extracted IOCs
Use Case
SOC triage, awareness validation, incident intake enrichment

Example Signals

  • Sender domain vs reply-to / return-path mismatch
  • Link text vs destination mismatch
  • Newly registered / suspicious domains
  • Credential-harvest phrasing patterns
