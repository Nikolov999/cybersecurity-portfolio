AI Digital Footprint Mapper

A desktop application for analyzing digital footprints, internal network exposure, and risk levels using a modern GUI and a local LLM (Ollama). The workflow is designed for internal lab use: safe experimentation, automated enrichment, and security insights without sending data to external services.

Project YAML Ingestion Internal Network Scan Nmap Integration Risk Scoring Graph Visualization Local LLM (Ollama) Ubuntu Defender VM

What It Delivers

  • Loads a YAML footprint (people, emails, domains, services, IPs, ownership links) into an interconnected asset graph.
  • Scans internal/lab ranges via Nmap and converts results into assets (hosts, ports, services, versions).
  • Applies heuristic risk scoring with tags explaining why an asset is risky.
  • Visualizes relationships and exposure using a built-in graph viewer.
  • Uses a local LLM assistant panel for offline recommendations and summarization.

Risk Scoring Signals

  • Service type and exposure level
  • Authentication weakness indicators
  • Protocol age / insecurity
  • Asset value weighting (lab model)
  • Risk tags for explainability

Workflow

1) Load Footprint (YAML)

Ingest structured footprint data and build the relationship graph.

  • People, emails, domains, socials
  • IPs, services, ownership links
  • Graph node/edge generation

2) Scan Internal Network

Run Nmap scans and convert results into assets automatically.

  • Hosts + open ports
  • Service names + versions
  • Exposure mapping

3) Analyze and Visualize

Score risk, filter by threshold, and explore relationships.

  • 0–100 score + tags
  • Asset table + filters
  • Graph visualization
