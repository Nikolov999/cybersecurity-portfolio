Windows Service Remote Creation Case Study

Remote service creation used for persistence and execution. Detection engineered through correlated telemetry: service install, registry modification, payload staging, and execution tracing.

Purple Team Windows Services sc.exe Registry Persistence Sysmon Detection Engineering

Key Findings

  • Remote sc.exe create installs a service.
  • ImagePath written to Services registry key.
  • Payload staged under C:\ProgramData\svc\.
  • Service start triggers payload execution.
  • High-confidence detection via event correlation.
