Detection Logic (Rule Ideas)

Rule A — WinRM-driven Scheduled Task Creation

Trigger when Sysmon EID 1 where Image ends with schtasks.exe and CommandLine contains /create, with ParentImage containing wsmprovhost.exe.

Escalate severity if command line contains /ru SYSTEM and trigger type is /sc ONLOGON or /sc ONSTART.

Rule B — Task Artifact Written

Trigger when Sysmon EID 11 writes under C:\Windows\System32\Tasks\. Add scoring when task naming mimics OS/update tooling.

Capture TargetFilename and adjacent Sysmon 1 process context for who wrote it.

Rule C — Immediate Execution After Creation

Trigger when Sysmon EID 1 includes /run and the task name matches a recent /create within a short window.
