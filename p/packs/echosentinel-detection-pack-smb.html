EchoSentinel Detection Pack — SMB Credential Abuse

Targeted detection for credential reuse and lateral movement using built-in Windows telemetry collected on the impact host. Detection is based on correlation, not single noisy events.

Listed Queries Rules SMB T1021.001

Telemetry Coverage (Impact Host)

  • Security: 4624 (LogonType 3), 4672 (Privileged token)
  • Service install: 4697 (Security) and/or 7045 (System)
  • Correlation model: auth → privilege → service creation within short timeframe

Extraction Anchors

Auth
4624 LogonType 3 + source IP attribution
Privilege
4672 pairing for elevated sessions
Impact
4697/7045 service creation as execution/persistence signal
Confidence
Correlation within 5-minute window

Detection Design

Correlation Signals

  • Signal 1: Network authentication (4624, LogonType 3)
  • Signal 2: Privileged token assignment (4672)
  • Signal 3: Service creation (4697 / 7045)

Individually benign. Together, within a short timeframe, indicates credential reuse and lateral movement.

Rule Rationale

  • Network logons alone are common and noisy
  • Privileged tokens occur in normal administration
  • Service creation can be legitimate

Correlation within ~5 minutes yields high confidence without requiring SMB file access visibility.

Key Fields to Extract

Security 4624 (LogonType 3)

TargetUserName, TargetDomainName, SubjectUserName, LogonType, LogonId, IpAddress/WorkstationName, AuthenticationPackageName.

Security 4672 + Service Install (4697/7045)

4672: SubjectUserName, SubjectDomainName, LogonId.
4697/7045: ServiceName, ServiceFileName/ImagePath, StartType, Account context (where present).

Validation Checklist

Pass Criteria

  • 4624 LogonType 3 captured on impact host
  • 4672 captured for same window/user context
  • 4697 or 7045 observed after auth + privilege
  • EchoSentinel alert built from correlation (not single event)

Constraints (Documented)

Client-side SMB file access telemetry may be unavailable due to endpoint limitations. Detection focuses on impact-side signals for realistic coverage.
